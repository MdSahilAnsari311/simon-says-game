# Simon Says Game (HTML • CSS • JavaScript)

- A fun and interactive Simon Says Memory Game built using pure JavaScript, CSS, and HTML.
- The player must remember and repeat the color sequence shown by the game.
- Each correct level increases the sequence length and makes the game more challenging.

# Features
- Automatically starts when user presses any key.
- Game generates a random color sequence every level.
- User must click buttons in the same order to continue.
- Flash animations for:
1. Game sequence
2. User clicks
- Game Over screen with:
1. Final Score
2. New High Score message
- Clean and colorful UI using CSS.
- Fully responsive and works on browser with no libraries.

# Technologies Used

- HTML – Game layout and structure.
- CSS – Button colors, animations, and page styling.
- JavaScript –
1. Level generation
2. Random color logic
3. Event listeners for buttons
4. Flash animations
5. Checking user input
6. High score logic
7. Game reset

# How to Play
1. Open the project in any web browser.
2. Press any key to start the game.
3. Watch the color that flashes → this is the game sequence.
4. Click the buttons in the same order.
5. Each correct level adds a new color to the sequence.
6. Game ends when you click the wrong color.
7. Restart by pressing any key again.
